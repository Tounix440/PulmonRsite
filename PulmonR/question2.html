Comment télécharger l'application Pulmon'R

Pour télécharger l'application Pulmon'R, suivez les étapes ci-dessous :

1. Rendez-vous sur le site officiel de Pulmon'R : www.pulmonr.com

2. Accédez à la page de téléchargement de l'application.

3. Choisissez la version de l'application compatible avec votre appareil (iOS ou Android).

4. Cliquez sur le lien de téléchargement et suivez les instructions qui s'affichent à l'écran.
